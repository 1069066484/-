使用方法：运行panorama_bg/panorama_bg/run.sh
其中“--basis_bin”参数是可执行的basis压缩程序，
    Windows上用../../../gView/basis/basis_universal/bin/Release/basisu.exe
    Linux上用../../../basis_prj/basis_universal/bin/basisu
    如果出错需要再在目标平台编译一遍basis项目。

在网页http://xxx.xxx.xxx.xxx/PanoramaDisplay_Demo/中可以打开并浏览全景图片，全景图片支持.jpg等常规图像格式，也支持.basis的压缩格式。
